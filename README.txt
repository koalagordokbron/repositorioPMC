	APLICACIÓN MINI CALCULADORA - PROYECTO DOCUMENTACIÓN JAVADOC

---------------------------------------------------------------------------------------------------------------

- Descripción: Aplicación creada en Java que calcula una raíz cuadrada y el valor absoluto de un número

- Versión: 1.0

- Estado de la aplicación: FINALIZADA

- Requisitos:
	- Tener instalado un IDE capaz de compilar Java
	- Tener instalado JDK 21
	- Tener instaldo JRE 21

- Tecnologías: NetBeans 20

- ¿Cómo utilizar la aplicación?
	Abrir el proyecto con el IDE deseado y pulsar "Run" o "F6"
